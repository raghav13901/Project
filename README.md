# SupDB


## Tech Stack
- Frontend: HTML/CSS/Bootstrap/ReactJS
- Backend: NodeJS/MongoDB

## Quick Start

- Fork and Clone the repo using

git clone https://github.com/MLSA-SRM/MLSA-Forms.git
cd SupDB

- Install dependencies using

npm install

- Run server using

npm run dev

- Then in Browser

localhost:3000
